# VaultX Privacy Policy Template

VaultX is a local-first secure notes application. The initial version does not create user accounts, does not use cloud storage, and does not transmit notes, attachments, passwords, biometric data, or vault metadata to external servers.

## Data Stored Locally

- Encrypted notes and note metadata
- Encrypted file attachments and voice recordings
- Encrypted local backups exported by the user
- Security settings, lockout state, and activity logs
- Intruder detection logs and encrypted selfie evidence when failed PIN attempts trigger capture

## Permissions

- Camera: intruder selfie capture after failed PIN unlock attempts
- Microphone: voice note recording
- Files: user-selected attachments and encrypted backup restore
- Biometrics: local authentication gate handled by the operating system

## User Controls

Users can export encrypted backups, restore matching encrypted backups, wipe all app data, configure decoy access, configure hidden vaults, and adjust lock timers.
